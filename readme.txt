Project Overview
This project is a React application that features a header, side navbar, and an internal report page displaying data in charts and tables. The application also includes a "No Data Available" page for other reports.

Features
1, Header and Side Navbar:

> A responsive header with a logo (sample image) and a side navbar.
> The side navbar contains links to "Internal Report" and other report pages.

2, Internal Report Page:

> Clicking on the "Internal Report" link directs the user to a static page with a detailed report.
> The report includes graphs and tables generated from the provided data.
> The "Reports" page is titled accordingly.

3, No Data Available Page:

> When clicking other report links, a "No Data Available" page is displayed.

4, Responsiveness:

> The application is designed to be fully responsive.
